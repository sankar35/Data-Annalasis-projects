# Business Data Analysis Project

Analyze 10,000+ rows of business data using SQL (SQLite) and Python (pandas, Excel-style) to find trends, patterns, and actionable business insights.

## Project Workflow

- Generate or use sample sales data (`src/generate_sample_data.py`)
- Analyze using pandas - groupby, summary statistics, trends
- Perform SQL analysis - using SQLite queries for more complex joins/groupings
- Document findings in the notebook/report

## To Run

```bash
pip install -r requirements.txt
cd src
python generate_sample_data.py
python data_analysis.py
python sql_analysis.py
```
